# CHANGE.md

> 系统层阶段变更模板。根目录使用 `CHANGE.md`，项目工作目录使用小写 `change.md`。

## 根 CHANGE.md

```markdown
# YYYY-MM-DD
## <项目 / 阶段 / 版本名称>
- 一句话：核心产出 / 方向调整 / 结构变化
```

## 项目 change.md

```markdown
# YYYY-MM-DD
## <Emoji> <核心产出>
> HH:mm | <意图背景>
- 变更文件:
  - A path/to/file
  - M path/to/file
  - D path/to/file
- 细节:
  - 要点
```

Emoji：`✨ Feat` · `🐛 Fix` · `📝 Docs` · `♻️ Refactor`
